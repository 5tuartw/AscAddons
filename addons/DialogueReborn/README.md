# Dialogue Reborn

Dialogue Reborn is a new Ascension 3.3.5 addon inspired by DialogueUI's presentation layer.

It does not attempt to port retail-only camera or frame systems. Instead, it builds a native Wrath-era replacement UI for:

- gossip conversations
- quest greeting lists
- quest detail screens
- quest progress screens
- quest reward screens

## Features

- custom immersive dialogue shell
- darkened screen overlay while talking to NPCs
- optional fading of action bars, minimap, chat, and buffs
- built-in keyboard shortcuts
- direct button activation for gossip and quest list entries to preserve Blizzard's hidden indices

## Commands

- `/dr`
- `/dr options`
- `/dr toggle`

## Notes

- This addon intentionally suppresses the visible Blizzard Quest and Gossip frames while its own UI is active.
- It should coexist with QuestKeys, but Dialogue Reborn provides its own keyboard handling while open.
- Camera changes are not implemented.
